# mini mediapad

a small macropad for hack club's blueprint. it has a screen for a keys-per-second tracker (and maybe bad apple in the future), four keys for rhythm games, a rotary encoder for volume, and prev/pause/next track buttons.

## Bill of Materials:
- 1x Seeed Studio XIAO RP2040
- 1x 0.91in OLED display (128*32)
- 7x Cherry MX-style Keyswitches
- 7x Blank DSA Keycaps
- 1x EC11 Rotary Encoder
- 4x M3x16mm screws
- 4x M3x5mx4mm heatset inserts
- 3D printed case, top and bottom
